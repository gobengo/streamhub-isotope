StreamHub-Isotope is a pluggable View for use with StreamHub-Backbone. It will use the jQuery Isotope plugin to lay out Content for a tiled display.

Example Usage

var app = new Hub({
    sdk: livefyreSdk,
    collection: {
        siteId: "303772",
        articleId: "prod0"
    },
    el: document.getElementById("example"),
    view: IsotopeView
}).start();

Using It

Bower is used for dependency management. You can install the dependencies with

bower install

StreamHub-Isotope is written as an AMD module. You will need to use an AMD loader like RequireJS to use it. Add it as a package in your RequireJS config:

packages: [{
    name: 'streamhub-isotope',
    location: './path/to/streamhub-isotope'
}]

Then you can use it like:

require(['streamhub-isotope'], function (IsotopeView) {
    // Hub some Streams, yo
})
