IsotopeView

new IsotopeView()

IsotopeView will render a Collection of Content as a tiled display using jQuery-Isotope
Parameters:
Name Type Description
opts.collection Collection A Collection of Content (see models/Collection)
opts.contentViewOptions Object Options to be passed to any Content Views this instantiates This is useful for passing custom templates for Content
opts.sources Object An object to configure stuff on a per-source basis Supports `twitter` and `rss` sub objects with the same opts as this root level
opts.isotope Object Options to be passed to isotope on instantiation
